Дано:
Вершина бинарного дерева определяется следующим образом:
    
    public class Node
    {
        public Node Left { get; private set; }
        public Node Right { get; private set; }
        public int Value { get; private set; }
    }

Бинарное дерево будем называть сбалансированным, если для каждой его вершины высота её двух поддеревьев различается не более чем на 1.

Задача:
Необходимо написать функцию bool CheckNodeIsBalanced(Node node), которая анализирует  заданное в аргументе бинарное дерево (с корнем node) и возвращает true в том и только том случае, если это дерево является сбалансированным.

Комментарий:
	• По возможности опишите самый оптимальный алгоритм анализа бинарного дерева.
	• Для решения задачи при необходимости можно описывать вспомогательные классы и функции.
